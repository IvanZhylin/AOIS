# AOIS Lab 1

Реализация лабораторной работы "Представление чисел в памяти компьютера".

## Что реализовано

- Перевод целых чисел в прямой, обратный и дополнительный коды (32 бита).
- Сложение и вычитание в дополнительном коде.
- Умножение и деление в прямом коде (деление с точностью до 5 знаков).
- Операции `+`, `-`, `*`, `/` для чисел с плавающей точкой в формате IEEE-754 binary32.
- Сложение чисел в `5421 BCD` (вариант `c`).
- Вывод в двоичном и десятичном виде для контроля.

## Принципы

- **KISS**: простые функции с одной задачей.
- **DRY**: общие bit-утилиты переиспользуются в модулях.
- **YAGNI/BDUF**: без лишней архитектуры и неиспользуемых абстракций.
- **SOLID/APO**: логика разделена по модулям (`bits`, `arithmetic`, `ieee754`, `bcd5421`).
